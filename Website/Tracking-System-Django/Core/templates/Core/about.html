Real Time Vehicle Tracking

This project aims to solve the problem of tracking and accountability of vehicles by providing a software platform. This project would serve as an important step to help in Vehicle tracking, component monitoring, vehicle analysis and fleet management. An efficient vehicle tracking system is implemented for monitoring of any equipped vehicle from any location at any time with the help of Global Positioning System (GPS) and Arduino Board which will enable users to locate their vehicles with ease and in a convenient manner. This web application will provide a dashboard for better and easy understanding of their vehicle positions and related statistics.
